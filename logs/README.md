# Logs

Use this directory for sanitized example logs.

Suggested layout:

```text
logs/
  2026-03-xx-good-stick/
    README.md
    spd-normal.txt
    spd-offline.txt
    pmic-normal.txt
    pmic-offline.txt
  2026-03-xx-bad-stick/
    README.md
```

Raw dumps are ignored by default. Commit only cleaned logs that are useful to future-you.
